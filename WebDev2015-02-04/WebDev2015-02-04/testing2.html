1. ID selector

Description:

The #id selector styles the element with the specified id.

Examples:

Style the element with id="firstname":

#firstname {

     background-color: yellow;

}



2. Element selector

Description:

The element selector selects all elements with the specified element name.

Examples:

Select and style all "h1" elements AND all "p" elements:

h1, p {

     background-color: yellow;

}



3. Class selector

Description:

The .class selector styles all elements with the specified class.

Examples:

Select and style all elements with class="example":

.example {

     background-color: yellow;

}



4. Attribute selector

Description:

The [attribute] selector is used to select elements with the specified attribute.

Examples:

Style all "a" elements with a target attribute:

a[target] {

     background-color: yellow;

}



5. Adjacet selector

Description:

The [attribute] selector is used to select elements with the specified attribute.

Examples:

Another use case might be styling of "caption span"s of the following "img" elements:

img + span.caption {

     font-style: italic;

}

6. Direct-descendent selector

Description:

The element element(Direct-descendent) selector is used to select elements inside elements.

Examples:

Select and style every "p" element that is inside "div" elements:

div p {

     background-color: yellow;

}



7. Grouped selectors

Description:

By combining the grouping of selectors that share the same declaration and declarations that share the same selector you can apply multiple declarations to multiple selectors. This technique allows you to create compact yet powerful CSS rules.

Examples:

combines Group Selectors with Group Declarations into one powerful technique:

body, th, td {font-size: 12px; font-family: arial, helvetica, sans-serif;

     background-color: yellow;

}



8. Descendent selector

Description:

A descendant selector is made up of two or more selectors separated by white space. A descendant selector of the form "A B" matches when an element B is an arbitrary descendant of some ancestor element A.

Examples:

Set a background color for all "ul" elements that are preceded by a "p" element with the same parent:

p ~ ul {

     background-color: yellow;

}



9. Universal selector

Description:

The * selector selects all elements. The * selector can also select all elements inside another element (See "More Examples").

Examples:

Select all elements, and set their background color to yellow:

* {

     background-color: yellow;

}



10. Pseudo classes

Description:

1. The :first-child pseudo-class matches an element that is the first child element of some other element.

2. The link pseudo-classes: :link and :visited

3. The dynamic pseudo-classes: :hover, :active, and :focus

4. The language pseudo-class: :lang

Examples:

a:link { color: red } /* unvisited links */

a:visited { color: blue } /* visited links */

a:hover { color: yellow } /* user hovers */

a:active { color: lime } /* active links */



11. Pseudo elements

Description:

Pseudo-elements behave just like real elements in CSS with the exceptions described below and elsewhere.

1. The :first-line pseudo-element applies special styles to the contents of the first formatted line of a paragraph.

2. The :first-letter pseudo-element must select the first letter of the first line of a block, if it is not preceded by any other content (such as images or inline tables) on its line.

3. The ':before' and ':after' pseudo-elements can be used to insert generated content before or after an element's content.

Examples:

P { font-size: 12pt; line-height: 1.2 }

P:first-letter { font-size: 200%; font-style: italic;font-weight: bold; float: left }

SPAN { text-transform: uppercase }
