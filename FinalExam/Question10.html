Question 10:

a: Only using one query statement, query the table for all users with empty (null) password or a password that has a length less than three characters.

b: Update any user's email address to be "andrew@bu.edu" when it was entered as asheehan@bu.edu.

c: Delete a user with the id of 12.

  • ANSWER:
  • a:[select * from CART where password is null and len(password) < 3]
  • b:[update CART set email = "andrew@bu.edu" where email = "asheehan@bu.edu"]
  • c:[delete from CART where user_id = 12]
