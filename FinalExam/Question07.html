Question:

Demonstrate how you would write a PHP function to use a default value when a value is not passed in?


Answer:

The following example shows we would write a PHP function to use use a default parameter. If we call the function setHeight() without arguments it takes the default value as argument:

Example:
<?php
function setHeight($minheight = 50) {
    echo "The height is : $minheight <br>";
}

setHeight(350);
setHeight(); // will use the default value of 50
setHeight(135);
setHeight(80);
?>
